# Change Log

All notable changes to the "Comment Craft" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [0.1.0] - 2024-01-XX

### Added

- Initial release of Comment Craft
- Generate comments for selected code blocks
- Format existing comments in documents
- Support for multiple comment styles (standard, JSDoc, JavaDoc, Doxygen)
- Language-aware comment generation
- Configurable comment styles and auto-formatting
- Context menu integration
- Keyboard shortcuts for quick access
- Command Palette integration

### Features

- Automatic comment generation based on code context
- Support for various programming languages
- Customizable comment styles
- Auto-formatting option
